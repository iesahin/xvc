# xvc pipeline list


## Synopsis

```console
$ xvc pipeline list --help
List all pipelines

Usage: xvc pipeline list [OPTIONS]

Options:
  -p, --pipeline-name <PIPELINE_NAME>  Name of the pipeline this command applies to
  -h, --help                           Print help

```
