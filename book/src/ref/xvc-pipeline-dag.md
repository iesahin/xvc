# xvc pipeline dag


## Synopsis 

```console
$ xvc pipeline dag --help
Generate mermaid diagram for the pipeline

Usage: xvc pipeline dag [OPTIONS]

Options:
  -n, --name <NAME>      Name of the pipeline this command applies to
      --file <FILE>      File to write the pipeline. Writes to stdin if not set.
      --format <FORMAT>  Format for graph
  -h, --help             Print help information

```
