# xvc pipeline update


## Synopsis 

```console
$ xvc pipeline update --help
Rename, change dir or set a pipeline default

Usage: xvc pipeline update [OPTIONS]

Options:
  -n, --name <NAME>        Name of the pipeline this command applies to
      --rename <RENAME>    rename this pipeline to
      --workdir <WORKDIR>  set the working directory to
      --set-default        set this pipeline default
  -h, --help               Print help information

```
