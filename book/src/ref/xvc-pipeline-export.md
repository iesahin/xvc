# xvc pipeline export


## Synopsis 

```console
$ xvc pipeline export --help
Export the pipeline to a YAML, TOML or JSON file

Usage: xvc pipeline export [OPTIONS]

Options:
  -n, --name <NAME>      Name of the pipeline this command applies to
      --file <FILE>      File to write the pipeline. Writes to stdin if not set.
      --format <FORMAT>  Format for output to stdout.
  -h, --help             Print help information

```
